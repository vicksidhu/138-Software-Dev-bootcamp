Responsive Containers & Breakpoints

Example 2: Working with Responsive Design

This example demonstrates Bootstrap's container types and breakpoint system.

Container Types

Bootstrap provides different container types for various layout needs.

Standard Container

.container - Fixed width container that changes max-width at each breakpoint

I'm inside a standard container

Try resizing your browser window to see how this container behaves

Fluid Container

.container-fluid - Full width container (100% width at all breakpoints)

I'm inside a fluid container

Responsive Containers

Containers that are 100% wide until a specific breakpoint

.container-sm - 100% width until small (576px) breakpoint

100% width until small breakpoint

.container-md - 100% width until medium (768px) breakpoint

100% width until medium breakpoint

.container-lg - 100% width until large (992px) breakpoint

100% width until large breakpoint

Breakpoints

Bootstrap uses breakpoints for responsive design.

Text Alignment at Different Breakpoints

This text changes alignment based on screen size:

Extra Small (default): Center aligned
Small (sm): Left aligned
Medium (md): Right aligned
Large (lg): Center aligned

Resize your browser to see the text alignment change

Visibility Control

Elements can be shown or hidden at different breakpoints:

This content only appears on medium screens (md) and larger
This content only appears on screens smaller than medium (md)
This content only appears on small (sm) and medium (md) screens
