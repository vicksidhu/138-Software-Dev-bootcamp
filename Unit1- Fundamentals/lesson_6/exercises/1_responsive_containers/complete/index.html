Bootstrap Breakpoint Tip

Bootstrap uses the pattern {property}-{breakpoint}-{value} for responsive utilities. Common breakpoints are: sm (≥576px), md (≥768px), lg (≥992px), and xl (≥1200px). For example, text-md-start means "align text left starting at medium screens". If no breakpoint is specified, the style applies to all screen sizes.
