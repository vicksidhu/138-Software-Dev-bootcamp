Example: Bootstrap Column Classes

Here's how to add column classes to create a responsive layout:

<div class="row">
  <div class="col-12 col-md-6 col-lg-3">
    Content goes here
  </div>
</div>

The classes mean:

  • col-12: Takes full width on mobile (12 columns out of 12)
  • col-md-6: Takes half width on medium screens (6 columns out of 12)
  • col-lg-3: Takes quarter width on large screens (3 columns out of 12)
