  - GitHub README files
  - Documentation
  - Forum posts

# Markdown Basics

| Element | Markdown Syntax |
|---------|----------------|
| Heading | `# H1, ## H2, ### H3` |
| Bold | `**bold text**` |
| Italic | `*italicized text*` |
| Link | `[title](https://www.example.com)` |
| Image | `![alt text](image.jpg)` |
| List | `- Item 1` <br> `- Item 2` |

# HTML Introduction

## What is HTML?
- HyperText Markup Language
- Foundation of web pages
- Uses elements/tags to structure content
- Creates the structure that browsers render

---

# HTML vs. Markdown

## Similarities
- Both are markup languages
- Both define document structure

## Differences
| Markdown | HTML |
|----------|------|
| Simple, limited formatting | Rich, extensive formatting |
| Easy to read raw | More verbose |
| Fewer tags | Many specialized tags |

---

# HTML Tags

## Basic Structure
```html
<tagname>Content goes here...</tagname>
```

## Common Elements
- Headings: `<h1>` to `<h6>`
- Paragraph: `<p>`
- Emphasis: `<em>`, `<strong>`
- Lists: `<ul>`, `<ol>`, `<li>`

# HTML Document Structure

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
    <!-- Metadata goes here -->
  </head>
  <body>
    <!-- Content goes here -->
  </body>
</html>
```

---

# HTML Document Components

## Understanding the Basic Structure
- `<!DOCTYPE html>`: Tells the browser this is an HTML5 document
- `<html>`: Root element containing all HTML content
- `<head>`: Contains metadata, title, linked resources
- `<body>`: Contains all visible content

*This structure is the foundation of every web page you'll create*
