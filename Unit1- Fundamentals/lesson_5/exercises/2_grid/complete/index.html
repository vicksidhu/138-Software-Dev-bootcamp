Exercise Instructions:

In this exercise, you'll create a responsive restaurant menu using CSS Grid.

  1. Transform the menu into a grid layout
  2. Create a responsive design that works on both mobile and desktop
  3. Use a mobile-first approach in your CSS
  4. Make the menu display in multiple columns on larger screens

Note: Edit only the CSS file. Do not modify the HTML.

Time: You have 10 minutes for this exercise.

Getting Started: Follow the TODO comments in the CSS file. Try to figure out the right CSS properties to use (you can reference the CSS Grid documentation if needed).
