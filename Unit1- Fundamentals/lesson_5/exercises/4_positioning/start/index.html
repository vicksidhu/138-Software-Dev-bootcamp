Exercise Instructions:

Use CSS Positioning to achieve the following:

  1. Make the navigation bar stick to the top of the page when scrolling using sticky positioning
  2. Position the price tag in each menu item at the top right corner using absolute positioning
  3. Position the badges (Popular, New, etc.) at the top left corner of the menu items using absolute positioning
  4. Create a promotional popup that appears in the bottom right corner of the viewport using fixed positioning
  5. Add a "back to top" button fixed at the bottom right corner using fixed positioning

Note: Edit only the CSS file. Do not modify the HTML.

Time: You have 20 minutes for this exercise.

Hint: Look for TODO comments in the CSS file for guidance.

Tip: Remember that elements need a positioned parent (position: relative) for absolute positioning to work relative to that parent.
