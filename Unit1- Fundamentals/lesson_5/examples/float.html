CSS Float Examples

The float property specifies whether an element should float to the left, right, or not at all.

Basic Float Examples

Float was originally designed to wrap text around images.

Float Left

This is an example of an image floated to the left. The text wraps around the right side of the image. Float was originally intended for this purpose - allowing content to flow around an element. Notice how the text continues to wrap until it clears the bottom of the floated element.

Additional text to demonstrate the wrapping behavior. The float property is commonly used with images to achieve this text-wrapping effect.

Float Right

This is an example of an image floated to the right. The text wraps around the left side of the image. This is useful when you want the image to appear on the right side of the content.

Additional text to demonstrate the wrapping behavior. The float property allows content to flow around an element rather than breaking to a new line.

No Float

This is an example with no float applied (or float: none). The image is displayed as a block element, and the text appears below it rather than wrapping around it.

The Clearfix Hack

Floated elements are removed from the normal flow, which can cause container collapse.

Container Collapse (Problem)

This container has two floated images but doesn't properly contain them. The container's height doesn't account for the floated elements.

Fixed with Clearfix (Solution)

This container uses the clearfix hack to properly contain the floated elements. The container's height now accounts for all the content inside it.

The clearfix hack uses ::after to add a hidden element that clears the floats: 
.clearfix::after {
    content: "";
    display: table;
    clear: both;
}
