CSS Positioning Examples

CSS positioning allows you to control the placement of elements on a web page.

Position: Static (Default)

Static positioning is the default. Elements are positioned according to the normal flow.
Static

This box has position: static. Offset properties (top, right, bottom, left) have no effect.

Position: Relative

Relative positioning positions an element relative to its normal position.
Relative

This box has position: relative; top: 30px; left: 30px;. It's offset from where it would normally be, but still occupies its original space in the flow.

Position: Absolute

Absolute positioning positions an element relative to its nearest positioned ancestor.
Absolute

The container has position: relative and the box has position: absolute; top: 40px; right: 40px;. The box is positioned relative to the container, not in the normal flow.

Position: Fixed

Fixed positioning positions an element relative to the viewport.

There is a pink box with position: fixed; bottom: 20px; right: 20px; that stays fixed at the bottom-right corner of the viewport as you scroll.

Fixed

Position: Sticky

Sticky positioning is a hybrid of relative and fixed positioning.

The purple box has position: sticky; top: 0;. It behaves like a relative-positioned element until it reaches the top of the viewport, then it "sticks" in place like a fixed-positioned element.

Z-Index Stacking

Z-index controls the stacking order of positioned elements.
z-index: 1
z-index: 2
z-index: 3

Higher z-index values appear in front of elements with lower z-index values.
