CSS Normal Flow Example

Normal flow is the default layout of HTML elements before any CSS positioning or layout techniques are applied.

Block-Level Elements

Block elements stack vertically and take up the full width available by default.
This is a block element. It takes up the full width of its container.
This is another block element. Notice how it appears below the previous element.
Yet another block element. Block elements always start on a new line.

Inline Elements

Inline elements flow horizontally and only take up as much width as necessary.
This is an inline element. This is another inline element. Yet another inline element. Inline elements flow horizontally. They don't create line breaks.
